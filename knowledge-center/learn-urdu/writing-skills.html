హోమ్› ఉర్దూ నేర్చుకోండి› రాయడం నేర్చుకోవడం
۳స్థాయి 3

ఉర్దూ రాయడం నేర్చుకోండి

لکھنے کی مہارت

Learn to write Urdu letters, words, and sentences in the elegant Nastaliq script.

అక్షరాలు రాయడం పదాలు రాయడం వాక్యాలు రాయడం
స్ట్రోక్ మార్గదర్శకం

నస్తాలీఖ్ స్ట్రోక్‌లు

నస్తాలీఖ్ రాయడం అభ్యాసం చేయడానికి ఈ ముఖ్యమైన స్ట్రోక్ నమూనాలు నేర్చుకోండి.

1

సమాంతర స్ట్రోక్

ب, ت, ث, ن, ی — The base of these letters is a flat or slightly curved line. Write right to left.

بـ تـ نـ یـ
2

వృత్తాకార స్ట్రోక్

م, ق, ف — These letters have a circular or oval shape. Draw counter-clockwise.

مـ قـ فـ
3

వంగిన స్ట్రోక్

ع, غ, ح, خ — These letters feature smooth, inward-curving strokes.

عـ غـ حـ خـ
4

నిలువు స్ట్రోక్

ل, ک, گ — Vertical drop combined with a base stroke. Practice each component.

لـ کـ گـ
5

నుక్తా (చుక్కలు)

Always add dots after completing the main stroke — small, round, and precisely placed.

ب ت ث ن پ چ
6

నస్తాలీఖ్ వాలు

Letters cascade diagonally downward right-to-left within each word — the signature Nastaliq look.

رحمت

For practice: 15 minutes daily works well. Always start with individual letters, then words, then sentences.

పద రాయడం

ఉర్దూ పదాలు రాయడం అభ్యాసం

ఈ ముఖ్యమైన ఇస్లామిక్ పదాలు రాయడం అభ్యాసం చేయండి. ప్రతి పదంలో అక్షరాల కలయికను గమనించండి.

اللہ
ا + ل + ل + ہ
Allah — God
رحمت
ر + ح + م + ت
Rahmat — Mercy
محبت
م + ح + ب + ت
Muhabbat — Love
سلام
س + ل + ا + م
Salaam — Peace
قرآن
ق + ر + آ + ن
Quran — The Quran
نماز
ن + م + ا + ز
Namaz — Prayer
ایمان
ا + ی + م + ا + ن
Iman — Faith
صبر
ص + ب + ر
Sabr — Patience
زندگی
ز + ن + د + گ + ی
Zindagi — Life
خدا
خ + د + ا
Khuda — God
تقویٰ
ت + ق + و + یٰ
Taqwa — God-consciousness
توبہ
ت + و + ب + ہ
Tawbah — Repentance
వాక్య రాయడం

ఉర్దూ వాక్యాలు రాయడం

ఈ ముఖ్యమైన ఉర్దూ వాక్యాలు రాయడం అభ్యాసం చేయండి — ప్రతి రోజు ఇస్లామిక్ జీవితంలో వాడే వాక్యాలు.

بسم اللہ الرحمٰن الرحیم
Bismillāh ir-Raḥmān ir-Raḥīm
ప్రతి మంచి పని మొదలుపెట్టే ముందు
الحمد للہ رب العالمین
Al-Ḥamdu lillāhi Rabbil ʿĀlamīn
అన్ని ప్రశంసలు అల్లాహ్‌కు
لا اِلٰہَ اِلَّا اللّٰہُ مُحَمَّدٌ رَّسُوْلُ اللّٰہِ
Lā ilāha illallāh Muḥammadur rasūlullāh
The Kalimah — declaration of faith
علم حاصل کرنا ہر مسلمان پر فرض ہے
Ilm ḥāsil karnā har Musalman par farz hai
జ్ఞానం సంపాదించడం ప్రతి ముస్లిం పై ఫర్జ్
اللہ ہر چیز پر قادر ہے
Allah har cheez par qadir hai
అల్లాహ్ ప్రతి విషయంపై సర్వశక్తిమంతుడు
صبر ایمان کا آدھا حصہ ہے
Sabr imān kā ādhā ḥissa hai
సబర్ ఈమాన్‌లో సగం భాగం

To improve: write one sentence five times daily. Start slow, then increase speed. Use a fine-tipped pen for Nastaliq practice.
