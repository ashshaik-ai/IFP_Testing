# docs/prompts/ — Reusable Prompts

Copy-paste prompt templates for common tasks on this repo. Each encodes the project's rules so a fresh Claude session does the task correctly. Always pair with [../../CLAUDE.md](../../CLAUDE.md) and [../../PROJECT_RULES.md](../../PROJECT_RULES.md).

| Prompt | Use when you want to… |
|---|---|
| [add-new-page.md](add-new-page.md) | Create a new page from the existing template/patterns |
| [add-bilingual-content.md](add-bilingual-content.md) | Add/edit Telugu+English copy correctly (right i18n system) |
| [add-lesson.md](add-lesson.md) | Add an Arabic/Urdu lesson with correct RTL |
| [change-design-token.md](change-design-token.md) | Change a colour/font/nav-height across all pages |
| [seo-pass.md](seo-pass.md) | Bring a page (or the site) up to the SEO standard |
| [qa-review.md](qa-review.md) | Run the full pre-merge QA checklist |
| [translate-card-bodies.md](translate-card-bodies.md) | Finish the student-guidance `CB_TE[]` body translation |

> Convention: start every task by stating which page(s) and which **localization system** (A or B) apply, then follow the [Definition of done](../../PROJECT_RULES.md#7-definition-of-done-qa-checklist).
